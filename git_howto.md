# Подсказка по GIT

**Создание репозитория**:

*Инициализация локального репозитория*
```sh
git init
```
*Получить информацию от Git о его текущем сотоянии*
```sh
Git Status
```
*Добавление файлов к следующему коммиту*
```sh
git add
```

```sh
git commit - "Massage"
```
```sh
git log
```
```sh
git log --oneline
```
```sh
git checkout
```
